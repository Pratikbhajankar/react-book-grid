# enovos
clone project 
# step 1. start express server
     cd express-server
     npm install
     npm run start
     
     server will be accesible on http://localhost:4040/
# step 2. start redux client 
     move back to enovos
     cd redux-client
     npm install
     npm serve-dev
     
     
     client will be accesible http://localhost:8082/
